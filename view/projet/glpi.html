Installation de GLPI

GLPI (Gestionnaire Libre de Parc Informatique) est un logiciel de gestion de parc informatique et de helpdesk open source. Voici les étapes pour installer GLPI sur un serveur local à l'aide de XAMPP.

Installation de GLPI

Prérequis

Avant de commencer l'installation de GLPI, assurez-vous d'avoir les éléments suivants :

- XAMPP installé sur votre machine.
- Le dossier GLPI téléchargé depuis le site officiel.



Étapes d'installation

Préparation des fichiers

Décompressez le fichier GLPI téléchargé dans le répertoire htdocs de votre installation XAMPP.
Ce répertoire se trouve généralement dans C:\xampp\htdocs sur Windows ou /opt/lampp/htdocs sur Linux.

Assurez-vous que le dossier GLPI est directement accessible depuis l'URL de votre serveur local.
Par exemple, si votre serveur local est accessible à l'adresse http://localhost, le dossier GLPI devrait être accessible à l'adresse http://localhost/glpi.



Configuration de la base de données

1 Ouvrez votre navigateur web et accédez à l'interface de gestion de MySQL de XAMPP en naviguant vers http://localhost/phpmyadmin.

2 Créez une nouvelle base de données pour GLPI (par exemple, glpi_db).

3 Assurez-vous d'avoir un utilisateur avec des privilèges suffisants pour accéder à cette base de données. Si nécessaire, créez un nouvel utilisateur et accordez-lui les privilèges nécessaires sur la base de données GLPI.



Installation via l'interface web

1 Ouvrez votre navigateur web et accédez à l'URL où vous avez placé les fichiers GLPI (par exemple, http://localhost/glpi).

2 Suivez les étapes de l'assistant d'installation de GLPI.

3 Lorsqu'on vous demande les informations de base de données, utilisez celles que vous avez configurées précédemment (nom de la base de données, nom d'utilisateur, mot de passe).

4 Terminez l'installation en suivant les instructions à l'écran.



Finalisation

1 Une fois l'installation terminée, assurez-vous de supprimer le dossier install du répertoire GLPI pour des raisons de sécurité.

2 Connectez-vous à l'interface administrateur de GLPI en utilisant le nom d'utilisateur et le mot de passe par défaut (généralement glpi/glpi).

3 Vous pouvez maintenant commencer à configurer GLPI selon vos besoins.



Conclusion

Félicitations ! Vous avez maintenant installé avec succès GLPI sur votre serveur local.
Vous pouvez utiliser ce puissant outil de gestion de parc informatique pour suivre vos ressources matérielles et logicielles, gérer les tickets d'assistance, et bien plus encore.
